# 圖片放置說明

## AIX 黏土課程海報

請將海報圖片放置於此資料夾中：

**檔案路徑:** `images/portfolio/aix-clay-course.jpg`

### 圖片規格建議

- **檔案格式:** JPG 或 PNG
- **建議尺寸:** 1600 × 1000 像素（16:10 比例）
- **檔案大小:** 建議小於 500KB（優化網頁載入速度）
- **檔案名稱:** `aix-clay-course.jpg` 或 `aix-clay-course.png`

### 如何準備圖片

1. 從您提供的海報截圖中裁切出主要內容
2. 調整尺寸為 1600×1000 像素
3. 儲存為 JPG 格式（品質 80-90%）
4. 重命名為 `aix-clay-course.jpg`
5. 放置於 `images/portfolio/` 資料夾中

### 替代方案

如果您希望使用不同的檔案名稱或格式，請同時更新以下檔案中的圖片路徑：

- **index.html** (第 110 行)
- **steam.html** (第 143 行)
